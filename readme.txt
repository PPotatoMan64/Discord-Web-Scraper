Scrapes the internet for a desired query, and uses a discord bot for ease of use.




How to use:

1. Download

2. Go to https://discord.com/developers and create an app. Make sure it has administrator permissions and can read messages.

3. Edit the python code, scroll all the way to the bottom, and replace ("Token_Here") with your bot token [ Should look something like this ("MTM2MzgwNTc2MzE...") ]

4. Run the python program.

5. Type !Help for the command list.
